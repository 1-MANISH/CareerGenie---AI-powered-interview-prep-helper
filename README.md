# 🚀 CareerGenie  – GenAI Career Intelligence Platform

> See Yourself Through Recruiter's Eyes

CareerGenie AI is a Generative AI-powered career intelligence platform that analyzes resumes and job descriptions to identify **skill gaps, strengths, and weaknesses**. The system uses **Gemini AI** to generate **technical and behavioral interview questions**, **personalized preparation plans**, and **job-specific optimized resumes**.

This platform helps job seekers prepare efficiently and improve their chances of getting hired.

---

## ✨ Features

### 📄 Resume Analysis
- Upload Resume (PDF/DOC)
- Parse Resume Content
- Extract Skills and Experience

### 🎯 Job Description Analysis
- Input detailed Job Description
- Identify Required Skills
- Match Resume with Job Description

### 🧠 AI Evaluation (Gemini)
- Skill Gap Detection
- Strength Analysis
- Weakness Detection
- Match Score Calculation

### 💡 AI Interview Preparation
- Technical Questions Generation
- Behavioral Questions Generation
- Question Intent Explanation
- Personalized Preparation Plan

### 📝 AI Resume Generator
- Job-specific Resume Creation
- ATS Friendly Resume
- Optimized Keywords

### 📊 Insights Dashboard
- Match Score Visualization
- Skill Gap Summary
- Improvement Suggestions
